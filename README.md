# 💰 Expense Tracker

A simple and responsive expense tracking application that helps users manage their daily expenses, track spending habits, and stay within their monthly budget.

## ✨ Features

- ➕ Add expenses
- ✏️ Edit expenses
- 🗑️ Delete expenses
- 📂 Categorize expenses
- 📅 Monthly expense summaries
- 📊 Expense visualizations using charts
- 💰 Set and monitor monthly budget limits
- 💾 Data persistence using Local Storage

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- Local Storage API
